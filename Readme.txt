Extract the Halcyon directory from the zip
Compile the code using the command 'make build' from within the top level of the Halcyon directory
Run the main game using the command 'make run' from within the top level of the Halcyon directory
Run the test cases using the command 'make run tests' from within the top level of the Halcyon directory
The document Test Document.pdf in the top level of the Halcyon directory explains in detail manual testing and JUnit testing
